# 빌드 툴링 결정 - Vite / pnpm / 프로젝트 구조 정비

- **생성일**: 2026-03-15
- **상태**: backlog
- **우선순위**: medium

## 내용

현재 프로젝트가 tsc + npm을 쓰고 있는데, 다음 사항에 대한 결정이 필요:

- Vite를 번들러로 도입할 것인가 (현재 tsc만 사용)
- pnpm으로 패키지 매니저를 전환할 것인가
- 전체 빌드/개발 구조를 더 정확하게 잡을 필요성

## 현재 상태

- 패키지 매니저: npm
- 빌드: tsc (TypeScript 컴파일만)
- 번들러: 없음
- 개발 실행: tsx

## 판단 포인트

- CLI 배포 시 번들링이 필요한가 (단일 파일 배포 vs 여러 파일)
- pnpm의 strict dependency resolution이 이 프로젝트에 이점이 있는가
- Vite의 빌드 속도/설정 이점이 현재 규모에서 의미 있는가
