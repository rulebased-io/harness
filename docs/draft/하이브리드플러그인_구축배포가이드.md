## 🏗️ 프로젝트 구축 단계

### Step 1: 프로젝트 초기화

```bash

cd harness-hybrid-plugin

# npm 초기화
npm init -y

# 의존성 설치
npm install @anthropic-ai/sdk @anthropic-ai/claude-code mcp @modelcontextprotocol/server-stdio
npm install --save-dev typescript @types/node @types/node
```

### Step 2: TypeScript 설정

```bash
# tsconfig.json 생성
npx tsc --init
```

## 🚀 배포 방법

### 방법 1: 로컬 테스트

```bash
# 빌드
npm run build

# 플러그인으로 설치
/plugin install ./harness-hybrid-plugin

# MCP 테스트
npm run mcp:inspect
```

### 방법 2: GitHub에 퍼블리시

```bash
# 1. GitHub 리포지토리 생성
# 2. 코드 푸시
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/harness-hybrid-plugin.git
git push -u origin main

# 3. 태그 추가 (버전 표시)
git tag -a v1.0.0 -m "Release v1.0.0"
git push origin v1.0.0

# 4. Claude Code에서 설치
/plugin install github:yourusername/harness-hybrid-plugin
```

### 방법 3: NPM 패키지로 퍼블리시

```bash
# npm 로그인
npm login

# 퍼블리시
npm publish

# 그 후 설치
npm install -g harness-builder-hybrid
```

- CHANGELOG.md: 버전 히스토리

### 3. 버전 관리

```bash
# 버전 업데이트
npm version patch   # 1.0.0 → 1.0.1
npm version minor   # 1.0.0 → 1.1.0
npm version major   # 1.0.0 → 2.0.0

# 퍼블리시
npm publish
```

### 4. CI/CD 설정

**GitHub Actions (.github/workflows/publish.yml):**
```yaml
name: Publish

on:
  push:
    tags:
      - 'v*'

jobs:
  publish:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: actions/setup-node@v2
        with:
          node-version: '18'
      - run: npm install
      - run: npm run build
      - run: npm publish
        env:
          NODE_AUTH_TOKEN: ${{ secrets.NPM_TOKEN }}
```

## 🚨 문제 해결

### 빌드 오류

```bash
# TypeScript 타입 검증
npx tsc --noEmit

# 의존성 재설치
rm -rf node_modules package-lock.json
npm install
```

### 플러그인 설치 실패

```bash
# 파일 권한 확인
chmod +x dist/hybrid-server.js

# plugin.json 검증
npx jsonlint .claude-plugin/plugin.json
```

### MCP 연결 실패

```bash
# 로그 확인
npm run mcp:inspect

# 포트 확인
lsof -i :5173
```
