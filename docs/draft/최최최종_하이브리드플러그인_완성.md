```

      "args": ["dist/hybrid-server.js"]
    }
  }
}

사용: MCP 도구 사용
```

## 💡 **핵심 기능 (5가지)**

모든 방식에서 **즉시 사용 가능**:

```
1. /harness-init (또는 init_harness)
   → 프로젝트 초기화

2. /test-case (또는 generate_test_case)
   → 테스트 케이스 생성

3. /integration-tests (또는 generate_integration_suite)
   → 통합 테스트 생성

4. /performance-test (또는 generate_performance_harness)
   → 성능 테스트 생성

5. /harness-config (또는 get_harness_config)
   → 설정 조회
```

## 🔥 **가장 강력한 사용 방법**

### Step 1: 플러그인 설치
```
/plugin install github:yourusername/harness-hybrid-plugin
```

### Step 2: MCP도 추가 (선택)
```json
설정 파일:
{
  "mcpServers": {
    "harness-builder": {
      "command": "node",
      "args": ["/path/to/dist/hybrid-server.js"]
    }
  }
}
```

### Step 3: 상황에 맞게 사용
```
빠른 작업:
/harness-init

복잡한 작업:
MCP 도구 사용

설정 변경:
설정 UI
```

```
→ 빠른 작업은 슬래시 커맨드
→ 복잡한 작업은 MCP 도구
```

### 패턴 3: 엔터프라이즈
```

→ Claude Code에서 플러그인
→ LangChain에서 MCP 도구
→ 자동화 도구에서 MCP 통합
```
