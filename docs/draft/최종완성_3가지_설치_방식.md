## 🎯 3가지 설치 방식 모두 준비됨!

### 1️⃣ **Claude Code 플러그인** ⭐ (가장 쉬움)
```
/plugin install github:yourusername/harness-plugin-complete
```

**특징:**
- ⭐ 가장 간단한 설치
- ⭐ 슬래시 커맨드 자동 지원
- ⭐ 설정 UI 제공
- ⭐ 상태 바 표시

**사용:**
```
/harness-init my-project
/test-case LoginTest "로그인"
/integration-tests user-crud
/performance-test api 100
/harness-config
```

### 2️⃣ **MCP 서버** (가장 유연)
```
{
  "mcpServers": {
    "harness-builder": {
      "command": "npx",
      "args": ["-y", "harness-builder-mcp"]
    }
  }
}
```

**특징:**
- 다양한 클라이언트 지원
- 완전한 MCP 프로토콜
- 더 강력한 기능

### 3️⃣ **Slash Commands** (가장 빠름)
```
/harness-init my-project jest typescript
```

**특징:**
- 매우 빠른 실행
- 간단한 인터페이스
- 직관적인 사용
